# MVP 定義

## MVP の目的
最小構成で学校・選手データの登録、参照、更新を成立させる。

## 含める機能

### 学校
- 学校作成
- 学校一覧
- 学校詳細
- 学校編集
- 学校削除（論理削除）

### 選手
- 学校詳細内での所属選手一覧表示
- 選手登録（手動）
- 選手詳細
- 選手編集

## 含めない機能
- 高度な検索
- OCR 自動取り込み
- 権限管理
- 選手削除
- 独立した選手一覧ページ

## MVP 完了条件
- 学校 CRUD が画面と API の両方から利用できる。
- 学校詳細から所属選手を参照できる。
- 選手の登録、詳細表示、編集が成立している。
- データ構造が `schema.sql` ベースで固定されている。

## 検討項目
- Phase2 以降に OCR 補助入力の導入を検討する。

## MVP外:
- OCR
- 高度検索
